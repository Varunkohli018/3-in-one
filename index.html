Basic HTML Knowledge

HTML (HyperText Markup Language) is the standard language for creating webpages. It uses tags to structure content.

Example:

<h1>Hello World</h1>
<p>This is a paragraph.</p>

Detailed HTML, CSS, and JavaScript Help

Common HTML Tags

Some frequently used HTML tags:

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
    <meta charset="UTF-8">
  </head>
  <body>
    <h1>Heading 1</h1>
    <h2>Heading 2</h2>
    <p>Paragraph</p>
    <a href="url">Link</a>
    <img src="image.jpg" alt="desc">
    <ul><li>List item</li></ul>
    <div>Division</div>
    <span>Span</span>
    <form>Form</form>
  </body>
</html>

Common CSS Properties

Here are some key CSS properties and their purposes:

color: value;           /* Text color */
background-color: value; /* Background color */
font-size: value;       /* Text size */
margin: value;          /* Outside spacing */
padding: value;         /* Inside spacing */
border: value;          /* Border style */
display: value;         /* Layout type (block, inline, flex, etc.) */
position: value;        /* Positioning (static, relative, absolute, fixed) */
width: value;           /* Element width */
height: value;          /* Element height */
flex: value;            /* Flexbox flexibility */
overflow: value;        /* Content overflow */

JavaScript Basics

JavaScript allows you to add interactivity:

// Selecting an element
const btn = document.getElementById('myButton');

// Adding an event listener
btn.addEventListener('click', () => {
  alert('Button clicked!');
});

// Variables
let name = "John";

// Functions
function greet() {
  console.log("Hello, " + name);
}
greet();
